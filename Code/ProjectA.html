Particle System Controls:

Common Controls:

W, A, S and D Keys: to move around
Up, Left, Down and Right Arrow Keys: to look around
Left Shift Key: Move in Vertical Direction - Up.
Left Control Key: Move in Vertical Direction - Down
r/R key: Interact with different systems by adding Random Force.
Space-bar: Single-Step.
p or P key: Pause/Resume.
c or C key: Toggle Streaks(clear-screen).
f or F: Toggle between different particle Systems

Bouncy Balls Controls:

You can change the Number of particles as well as the type of solver,
from the DAT.GUI Widget on the left of the screen. Press Reload after inputting the values.

Fire Particles Controls:

You can change the Number of particles as well as the type of solver,
from the DAT.GUI Widget on the left of the screen. Press Reload after inputting the values.

Tornado Particle Controls:

You can change the Number of particles as well as the type of solver,
from the DAT.GUI Widget on the left of the screen. Press Reload after inputting the values.

Springy Tetrahedron Particle Controls:

You can change a number of parameters here:
The Rest Length, Spring Coefficient (k) and the dampness Coefficient of a spring.
Along with that you can change the type of solver as well.

NOTE: THESE ONLY TAKE EFFECT IF THE RELOAD BUTTON IS PRESSED.

Boids Particle Controls:

There's an aggressor in the middle of the box defined by another box which
repels any particle which gets too close. Change the aggressor's settings from the
DAT.GUI Console.
